В папке "russian" предоставлены материалы, которые использовались в проекте
перевода, а именно:

Fontgen - простейший Java-скрипт, дешифрующий шестнадцатиричные значения и
отрисовывающий их попиксельно в таблице. Использовался для отрисовки 8x16 
русских шрифтов (были срисованы с DOSBox). В таблице буквы отображаются 
перевёрнутыми на 90 градусов против часовой стрелки, но по факту всё выглядит
должным образом. В данный момент уже не актуален, т.к. новая версия библиотеки
TextScreen использует шрифтовой PNG атлас.

Таблица символов - таблица с русским алфамитом в кодировке Windows-1251 и
ALT+символами для кодировки DOS CP866 (использовалась для экранов ENDOOM).
Дополнительно могут пригодиться:
http://www.ascii-codes.com/ и http://www.celitel.info/klad/tabsim.htm

--------------------------------------------------------------------------------

Вся работа с исходными кодами проводилась в среде Code::Blocks v16.01
Скачать можно по ссылке: http://www.codeblocks.org/

Для компиляции потребуются следующие библиотеки разработчика:

SDL2 2.0.5     - https://www.libsdl.org/download-2.0.php
SDL2_Mixer 2.0 - https://www.libsdl.org/projects/SDL_mixer/
SDL2_Net 2.0   - https://www.libsdl.org/projects/SDL_net/

* Это актуальные версии на момент выпуска проекта. Возможно, будут работать и
более новые версии библиотек.

Англоязычная инструкция по компиляции расположена в Chocolate Doom Wiki:
https://www.chocolate-doom.org

--------------------------------------------------------------------------------

Компиляция под Linux всё ещё возможна и абсолютно работоспособна, однако я всё
ещё не настолько хорошо знаю Linux, чтобы реализовать полноценную установку.

Чтобы получить работоспособную игру, скомпилируйте исходные коды:
./autogen.sh
make
sudo make install

По завершению процедуры поместите папку с русскоязычными ресурсами "russian"
(из архива с дистрибудитов для Windows), а также необходимые IWAD-файлы в
в /usr/local/bin/
